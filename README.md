# client-side
> a project for Blanja.com online test. Developed by Faishal Arif

## Build Setup

>Clone this repository
# Open clientSide folder
``` bash
$ cd clientSide

```

# install dependencies
``` bash
$ npm install # Or yarn install

```

# serve with hot reload at localhost:3000
``` bash
$ npm run dev

```
